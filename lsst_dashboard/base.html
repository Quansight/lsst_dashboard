class lsst_dashboard.base.Application(**params)¶

The Application class is the top-level object which coordinates the layout of the different components of the dashboard. It primarily consists of a header and body, the header is generated automatically from the current title and a logo while the body is a :class:`Component` which implements a :meth:`Component.panel` method which returns panel objects which can be composed into the overall application layout.

class lsst_dashboard.base.Component(**params)¶

Baseclass for visual components of the :class:`Application`. Defines a common API shared by all displayable components which allows access to the parent object, the :class:`Application` and defines methods which should return the title and a displayable panel object.
